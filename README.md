# Frontend Carrito de compras Pet Store
Proyecto desarrollado para estudio. 
Universidad Politécnico Grancolombiano
 
Tecnologías usadas:
* VueJS
* Typescript
* Vuetify


## Development
```sh
npm run serve
```

## Production
```sh
npm build
```

## Unit test
```sh
npm test:unit
```
